# n8n Webhook Production Setup

**Issue:** Test URL vs Production URL

---

## n8n Webhook Modes

**Test Mode:**
- URL: `https://.../webhook-test/...`
- Only visible in UI
- Manual testing only

**Production Mode:**
- URL: `https://.../webhook/...`
- Publicly accessible
- LINE can call

---

## How to Enable Production

1. Open workflow in n8n UI
2. Click webhook node
3. Find "Production" toggle
4. Turn ON
5. Copy production URL
6. Set in LINE Developers Console
