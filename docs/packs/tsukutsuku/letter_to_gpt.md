## P.S. - 実装上の懸念事項

君に特に検証してほしい点：

### 1. Instagram Graph API
- 2024年以降、自動投稿の規約が厳しくなった
- ビジネスアカウント必須
- 投稿頻度の制限
→ 本当に実装可能か確認してほしい

### 2. X API（旧Twitter）
- 有料プランが必須（$100/月〜）
- レート制限が厳しい
→ コスト的に見合うか検証

### 3. note API
- 公式APIが存在しない
- スクレイピングは規約違反の可能性
→ 代替手段を考えるべきか

### 4. AI APIコスト
- GPT-4o: $5/$15（入力/出力 per 1M tokens）
- Claude 3.5 Sonnet: $3/$15
- 1投稿あたりの平均トークン数
→ 月間コストの試算が必要

これらを君の視点で検証し、フィードバックをお願いします。
