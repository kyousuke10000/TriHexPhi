# Dual-Breath Protocol

## Concept
- **Round**: Intellectual/content evolution (e.g., Genesis Manuscript iterations)
- **Phase**: Technical/infrastructure tasks (e.g., Albedo integration)

## Usage
- `:::round[II]` – Round II tags
- `:::phase[Albedo]` – Phase tags
