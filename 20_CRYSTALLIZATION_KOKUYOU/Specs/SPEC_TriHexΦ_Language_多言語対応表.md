## 🔥 言語別：日本人の典型的弱点パターン

### 英語

```yaml
弱点パターン Top 5:
  
  1. 過去形・過去分詞（頻度: 極めて高）
     間違い例:
       - "I go to school yesterday" → went
       - "I have see the movie" → seen
     
     原因:
       日本語に時制変化がない
     
     対策:
       不規則動詞100個を集中暗記
       過去形ドリル（Bootstrap Memory活用）
  
  2. 冠詞 a/the/無冠詞（頻度: 極めて高）
     間違い例:
       - "I have dog" → a dog
       - "I went to park" → the park
       - "I like the music" → I like music
     
     原因:
       日本語に冠詞がない
     
     対策:
       可算/不可算名詞の理解
       定冠詞/不定冠詞のルール暗記
  
  3. 前置詞 in/on/at（頻度: 高）
     間違い例:
       - "I arrived to Tokyo" → in Tokyo
       - "at Monday" → on Monday
       - "in 3 o'clock" → at 3 o'clock
     
     原因:
       日本語の「に/で/を」との1対1対応がない
     
     対策:
       場所・時間の前置詞ルール暗記
  
  4. 単数/複数（頻度: 高）
     間違い例:
       - "I have two book" → books
       - "Informations" → Information（不可算）
     
     原因:
       日本語に単複の区別がない
     
     対策:
       可算/不可算名詞リスト暗記
  
  5. 語順（疑問文・否定文）（頻度: 中）
     間違い例:
       - "You are happy?" → Are you happy?
       - "I don't can go" → I can't go
     
     原因:
       日本語と語順が違う
     
     対策:
       基本語順パターン暗記
```

### 中国語

```yaml
弱点パターン Top 5:
  
  1. 声調（四声）（頻度: 極めて高）
     間違い例:
       - mā（妈）と má（麻）と mǎ（马）の区別
     
     原因:
       日本語は高低アクセント（中国語は声調）
     
     対策:
       音声認識で声調チェック
       シャドーイング集中練習
  
  2. 量詞（頻度: 高）
     間違い例:
       - "一人人" → "一个人"
       - "一本狗" → "一只狗"
     
     原因:
       日本語の助数詞と対応が違う
     
     対策:
       よく使う量詞50個暗記
  
  3. 了/過/着の使い分け（頻度: 高）
     間違い例:
       - "我去了学校" → "我去学校了"（文末）
       - "我看过这本书" → 過（経験）
     
     原因:
       日本語にない概念
     
     対策:
       用法パターン暗記
  
  4. 語順（時間+場所+動詞）（頻度: 高）
     間違い例:
       - "我在学校昨天学习" → "我昨天在学校学习"
     
     原因:
       日本語と語順が違う
     
     対策:
       基本語順パターン暗記
  
  5. 把構文（頻度: 中）
     間違い例:
       - "我吃了苹果" vs "我把苹果吃了"
     
     原因:
       日本語にない文法
     
     対策:
       把構文の使い方集中練習
```

### 韓国語

```yaml
弱点パターン Top 5:
  
  1. パッチム（頻度: 極めて高）
     間違い例:
       - 発音: ㄱ/ㅋ/ㄲ の区別
       - 連音化: "한국어" → ハングゴ（not ハングゴオ）
     
     原因:
       日本語にない子音
     
     対策:
       音声認識で発音チェック
       パッチム練習ドリル
  
  2. 尊敬語/謙譲語/丁寧語（頻度: 高）
     間違い例:
       - "食べる": 먹다/드시다/잡수시다
       - 使い分けミス
     
     原因:
       日本語より複雑
     
     対策:
       敬語レベル表暗記
  
  3. 助詞（이/가、은/는）（頻度: 高）
     間違い例:
       - パッチム有無で変わる
       - "책은" vs "책이"
     
     原因:
       日本語の「は/が」と微妙に違う
     
     対策:
       助詞ルール暗記
  
  4. 連音化・濃音化（頻度: 高）
     間違い例:
       - "먹어요" → モゴヨ（連音化）
       - "학교" → ハッキョ（濃音化）
     
     原因:
       日本語にない音韻変化
     
     対策:
       音韻変化ルール暗記
  
  5. 漢字語 vs 固有語（頻度: 中）
     間違い例:
       - "1つ" → 하나（固有語）vs 일（漢字語）
     
     原因:
       使い分けが複雑
     
     対策:
       よく使う数詞暗記
```

### フランス語

```yaml
弱点パターン Top 5:
  
  1. 男性名詞/女性名詞（頻度: 極めて高）
     間違い例:
       - "le table" → "la table"（女性）
     
     原因:
       日本語に性がない
     
     対策:
       単語とセットで性を暗記
  
  2. リエゾン（頻度: 高）
     間違い例:
       - "les enfants" → レザンファン（リエゾン）
     
     原因:
       日本語にない音韻現象
     
     対策:
       リエゾンルール暗記
  
  3. 動詞の活用（頻度: 高）
     間違い例:
       - avoir/être の活用ミス
     
     原因:
       日本語は活用が少ない
     
     対策:
       基本動詞100個の活用暗記
  
  4. R音（頻度: 高）
     間違い例:
       - "Paris" の R の発音
     
     原因:
       日本語にない音
     
     対策:
       発音練習（音声認識活用）
  
  5. 時制（複合過去/半過去）（頻度: 中）
     間違い例:
       - 使い分けミス
     
     原因:
       日本語にない時制区別
     
     対策:
       時制ルール暗記
```

## 💡 TriHexΦ Language での自動対応

```yaml
仕組み:
  各言語の「日本人の典型的弱点パターン」をデータベース化
  
  例（英語）:
    weaknesses_database/english/japanese_learners.yaml
    
    top_mistakes:
      - pattern: "past_tense"
        frequency: "extremely_high"
        examples:
          - "I go yesterday" → "I went yesterday"
        focus_drill: "irregular_verbs_100.md"
  
  あなたの間違いと照合:
    あなた: "I go to school yesterday"

      → past_tense パターン検知

  言語ごとの特性に自動対応
  日本人学習者に最適化された学習プラン
```
