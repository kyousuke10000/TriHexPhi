## 📋 Part 1: 不要なリポジトリの削除

### 手順1: リポジトリ一覧確認

```
1. GitHubにアクセス
   https://github.com/kyousuke10000?tab=repositories

2. 全リポジトリを確認
   → どれが必要？どれが不要？

3. 残すべきリポジトリ（Cursor推奨）:
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   ✅ TriHexPhi - メインプロジェクト（絶対残す）
   
   その他のリポジトリ:
   → しりゅうが判断してください
   → 「勉強で作った」ものは全部削除でOK
```

### 手順2: リポジトリ削除方法

```
安全な削除手順:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. GitHubで対象リポジトリを開く
   https://github.com/kyousuke10000/[リポジトリ名]

2. Settings（設定）をクリック

3. 一番下までスクロール
   「Danger Zone（危険ゾーン）」まで

4. 「Delete this repository」をクリック

5. 確認画面で以下を入力:
   「kyousuke10000/[リポジトリ名]」
   
6. 「I understand the consequences, delete this repository」
   をクリック

7. パスワード入力（または2FA認証）

8. 削除完了！
```

### 注意点

```
⚠️ リポジトリ削除は取り消せません
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

削除前にチェック:
✅ 本当に不要か？
✅ ローカルにバックアップあるか？
✅ 他の人が使ってないか？（コラボレーター）

安全な方法:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ 削除する前に「Archive」にする（非推奨だが）
✅ または Private にして隠す
✅ 確実に不要なら即削除でOK
```

## 📋 Part 2: ワークフロー失敗ログの管理（プロの判断）

### Cursorの推奨：失敗ログは残す（期限付き）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【プロの判断】失敗ログは残すべき
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

理由:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. デバッグの証拠
   ✅ なぜ失敗したか後から確認できる
   ✅ 同じエラーが再発した時の参考

2. 学習記録
   ✅ 「こういう失敗をした」という記録
   ✅ 成長の証

3. チーム共有
   ✅ 他のAIが同じミスを避けられる
   ✅ ナレッジベースになる

4. トラブルシューティング
   ✅ 問題が起きた時の調査に使える

5. 統計データ
   ✅ 成功率の計算
   ✅ 改善の可視化

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### ただし、期限を設ける

```
GitHub Actions のログ保持期限:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

デフォルト: 90日間

Cursor推奨:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ 成功ログ: 30日間保持（短め）
✅ 失敗ログ: 90日間保持（長め）
✅ 90日経過 → 自動削除

設定方法:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
GitHub Settings → Actions → General
→ "Artifact and log retention" を 90 days に設定
```

### 手動削除が必要な場合

```
どういう時に削除する？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

削除してOK:
✅ テスト用の実行（動作確認だけ）
✅ 意味のない失敗（タイポとか）
✅ 大量に失敗して見づらい時

削除しない方がいい:
❌ 原因不明の失敗（調査に必要）
❌ 本番デプロイの失敗（重要な証拠）
❌ セキュリティ関連の失敗

削除方法:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. GitHub → Actions タブ
2. 削除したいWorkflow Run をクリック
3. 右上の「...」メニュー
4. 「Delete workflow run」をクリック
5. 確認 → 削除
```
