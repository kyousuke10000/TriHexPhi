## 💎 Ⅵ. 「知らず知らずのうちに」の正体

### 意識 vs 無意識

```yaml
意識的な学習（失敗パターン）:
  「今日は過去形を勉強しよう」
  「単語を50個覚えよう」
  「文法を理解しよう」
  
  → 意識的
  → 義務感
  → 続かない

無意識の習得（成功パターン）:
  留学:
    「お腹すいた、何か食べたい」
    → "I want to eat"
    
    → 気づいたら100回言ってる
    
  恋愛:
    「会いたい」
    → "I want to see you"
    
    → 気づいたら1,000回言ってる
  
  ゲーム:
    「行こう！」
    → "Let's go!"
    
    → 気づいたら1万回言ってる
  
  = 無意識の反復
  = 楽しいから気づかない
  = でも回数は多い

「知らず知らずのうちに」の正体:
  楽しい・必要だから
  → 意識せず大量反復
  → 気づいたら話せてる
  
  = 無意識の大量反復
```
