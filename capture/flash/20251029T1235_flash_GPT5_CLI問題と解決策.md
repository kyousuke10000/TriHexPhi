**From**: しりゅう

**Date**: 2025-10-29 12:35  

---

## 🎯 問題

```yaml
しりゅうの発見:
  「GPTってCLIないよね。確か？」

真実:
  ✅ 正しい
  
  GPT-5（ChatGPT Plus）:
    - Web版のみ
    - CLIは公式に提供されていない
  
  でも、OpenAI API（GPT-4o）:
    - APIがある
    - CLIで使える
    
  問題:
    GPT-5（最新）はAPIでは使えない
    GPT-4oまで
```

---

## 💡 解決策（3つ）

### Option A: OpenAI API（GPT-4o）を使う

```yaml
方法:
  GPT-5の代わりに、GPT-4oをCLIで使う

利点:
  ✅ CLI対応
  ✅ API経由で自動化可能
  ✅ GitHub Actionsでも使える
  ✅ コスト効率良い（$0.0025/1K tokens）

欠点:
  ⚠️ GPT-5より性能は劣る
  ⚠️ でもGPT-4oでも十分強力

実装:
  npm install -g openai-cli
  
  使い方:
    echo "Phase 2の設計をレビューして" | openai

結論:
  これで十分かも
```
